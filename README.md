# Telegram Study Planner Bot

This bot can create a personalised plan for exam preparation for user.

## Technologies

* Aiogram
* Psycopg2
* Python-dotenv

## Functions

1. User registration
2. User can choose:
   * Subject of exam
   * Date of exam
   * Length of study session
3. Retrieve personalised plan for study
4. Ticking completed tasks
5. Send notifications
